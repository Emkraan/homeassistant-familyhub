# Changelog

## 2026.5.0-beta (2026-05-11)

Initial beta release.

- GUI config flow — set up from Settings → Devices & Services, no YAML required
- Camera entity exposing composite still image from all refrigerator cameras
- Local-only communication — no cloud, no Samsung account required
- Proper device registry entry with manufacturer and model
- ConfigEntryNotReady on startup if refrigerator is unreachable
